# Schwindt_3

## Overview

Schwindt_3 is a C# project built using the .NET Framework.  
This project demonstrates object-oriented programming principles, structured application design, and practical implementation of C# development concepts.

The goal of this project was to strengthen my understanding of application structure, debugging, and version control using Git.

---

## Technologies Used

- C#
- .NET Framework
- Visual Studio / Visual Studio Code
- Git & GitHub

---

## Project Structure

- `Program.cs` – Main application entry point
- `*.cs` – Core application logic
- `Schwindt_3.csproj` – Project configuration file

---

## How to Run the Project

### Requirements
- .NET SDK installed
- Visual Studio or Visual Studio Code

### Running from Command Line

```bash
dotnet build
dotnet run
```
